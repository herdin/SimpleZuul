## SimpleZuulTest project
- netflix zuul 을 이용한 다중 서비스의 api gateway 예제 프로젝트

### SimpleRestTypeA module
- 카드 서비스를 등록/수정/삭제 서비스
- 8090 포트

### SimpleRestTypeB module
- 카드 서비스를 조회 서비스
- 8091 포트

### SimpleZuul
- netflix zuul 을 사용한 api gateway
